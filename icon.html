Make your Belonging Icon

Answer a few questions to generate one icon. Download it or share a link.

No personal information is collected. The share link stores only icon settings.

Place — location / neighborhood / nature
Object — things / rituals / routines
Person — relationships / chosen family
Language — words / culture / identity
By default, no data will be submitted. Even if you check the box, it's currently just a demonstration: the payload will be printed to the console.
Share link will write parameters into the URL (no personal information).
